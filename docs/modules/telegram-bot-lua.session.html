Module telegram-bot-lua.session

per-chat/user session store with pluggable backends.

the default backend is in-memory; api.session.use(backend) swaps it for a redis- or db-backed one (a table with get/set/clear(self, key) methods).

Functions

api.session.clear(obj) clear the session for the chat/user.
api.session.get(obj) get the session table for the chat/user in an update.
api.session.key(obj) the stable session key (chat:user) for an update or payload.
api.session.memory() create an in-memory session backend.
api.session.set(obj, value) replace the session value for the chat/user.
api.session.use(backend) replace the session backend.
