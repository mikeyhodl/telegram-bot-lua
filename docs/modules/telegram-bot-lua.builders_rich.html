Module telegram-bot-lua.builders_rich

rich message builders (Bot API 10.1).

helpers for InputRichMessage payloads (sent) and the RichText / RichBlock structures received in message.rich_message. each returns a plain table.

Functions

api.input_rich_message(opts) InputRichMessage; supply exactly one of opts.html or opts.markdown.
api.input_rich_message_content(rich_message) wrap an InputRichMessage for inline/guest/web app query results.
api.link(url) a link object.
api.rich_block_animation(animation, opts) an animation block; opts: has_spoiler, caption.
api.rich_block_heading(text, size) section heading; size 1-6, 1 is largest.
api.rich_block_list_item(label, blocks, opts) a list item; opts: hascheckbox, ischecked, value, type.
api.rich_block_map(location, opts) a map block; opts: zoom, width, height, caption.
api.rich_block_photo(photo, opts) a photo block; photo is an array of photo sizes; opts: has_spoiler, caption.
api.rich_block_table(cells, opts) a table; cells is an array of arrays of cells; opts: isbordered, isstriped, caption.
api.rich_block_table_cell(text, opts) a table cell; opts: is_header, colspan, rowspan, align, valign.
api.rich_block_thinking(text) a thinking placeholder; only valid in sendrichmessage_draft.
api.rich_block_video(video, opts) a video block; opts: has_spoiler, caption.
api.rich_message(blocks, is_rtl) a received rich message from an array of blocks.
