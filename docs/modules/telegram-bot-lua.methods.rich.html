api.send_rich_message(chat_id, rich_message, opts)
send a rich formatted message to a chat. a rich message is described with HTML or markdown via an InputRichMessage object; see api.inputrichmessage for a builder.

Parameters:

  • chat_id number|string unique identifier for the target chat or username of the target bot/supergroup/channel
  • rich_message table|string an InputRichMessage object describing the message to send
  • opts table optional parameters
    • business_connection_id string unique identifier of the business connection
    • message_thread_id number unique identifier for the target message thread (topic) of a forum
    • direct_messages_topic_id number identifier of the direct messages topic to send to
    • disable_notification boolean send the message silently
    • protect_content boolean protect the contents of the sent message from forwarding and saving
    • allow_paid_broadcast boolean allow up to 1000 messages per second for a fee
    • message_effect_id string unique identifier of the message effect to add; private chats only
    • suggested_post_parameters table parameters of the suggested post to send
    • reply_parameters table description of the message to reply to
    • reply_markup table additional interface options

Returns:

    table,number the response object and HTTP status
api.send_rich_message_draft(chat_id, draft_id, rich_message, opts)
stream a partial rich message to a private chat as a draft. repeated calls with the same draftid animate the changes; useful for streaming AI-generated replies. a non-zero draftid is required.

Parameters:

  • chat_id number unique identifier for the target private chat
  • draft_id number unique identifier of the message draft; must be non-zero
  • rich_message table|string an InputRichMessage object describing the partial message
  • opts table optional parameters
    • message_thread_id number unique identifier for the target message thread

Returns:

    table,number the response object and HTTP status
